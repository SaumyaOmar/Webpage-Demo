Do you know what programming languages are?

A programming language is a formal language, a set of instructions that produce various kinds of output. Programming languages are used in computer programming to implement algorithms. Most programming languages consist of instructions for computers. Since the early 1800s, programs have been used to direct the behavior of machines such as Jaquard Looms, music boxes and player pianos. The programs for these machines (such as a player scrolls) did not produce different behavior in response to different inputs or conditions
